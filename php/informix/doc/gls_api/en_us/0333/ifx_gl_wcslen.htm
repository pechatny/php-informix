ifx_gl_wcslen - get wide character string length

SYNOPSIS


#include <ifxgls.h>
int ifx_gl_wcslen(gl_wchar_t *wcs);

DESCRIPTION

This function computes the number of wide-character codes in the wide character string to which wcs points, not including the null-terminating wide-character code.

RETURN VALUE

This function returns wcs; no return value is reserved to indicate an error.

ERRORS

No errors are defined.
