ifx_gl_mb_loc_max - maximum number of a bytes in a multi-byte character

SYNOPSIS

#include <ifxgls.h>
int ifx_gl_mb_loc_max(void);

DESCRIPTION

This function returns the maximum number of bytes that any multi-byte character can occupy.

RETURN VALUES

This function returns the maximum number of bytes that any multi-byte character can occupy.

ERRORS

None
