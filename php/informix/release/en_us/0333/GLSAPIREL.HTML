Description

IBM Informix GLS is an application programming interface (API) that provides functions and macros for the implementation of applications with multicultural support that are compatible with IBM Informix GLS globalization. This API allows applications to access information in IBM Informix locale files. The IBM Informix GLS 6.00 library incorporates the International Components for Unicode (ICU) 4.8.1 library.

For a complete list of what's new and changed in this release, go to http://www.ibm.com/support/knowledgecenter/SSGU8G_12.1.0/com.ibm.po.doc/new_features_ce.htm.
